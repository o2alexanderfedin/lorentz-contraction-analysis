-------------------------
Instructions for Use:
-------------------------

To compile the article to PDF:

1. Option 1: Use Overleaf (recommended)
   - Go to: https://www.overleaf.com
   - Create a new project
   - Upload 'lorentz_contraction_fedin.tex'
   - Press 'Recompile' to generate the PDF

2. Option 2: Compile locally
   - Requirements: LaTeX distribution (TeX Live, MikTeX, etc.)
   - In terminal, run:
       pdflatex lorentz_contraction_fedin.tex

Optional:
- Add your ORCID or institutional affiliation in the \author{} section before submitting to arXiv.
- Submit to arXiv under category: physics.class-ph or gr-qc

For any extensions (adding figures, bibliography), follow standard LaTeX procedures.

Enjoy!
